# MVP验收清单 V1.0

## 1. 身份与权限

- [ ] 用户只能访问当前Workspace数据；
- [ ] 停用用户立即失去访问；
- [ ] 销售、顾问、财务、数据管理员和管理员权限符合矩阵；
- [ ] 未成年人敏感字段无权限时不可通过API、导出或搜索获取；
- [ ] 导出、合并、软删除、恢复和敏感读取写入审计。

## 2. 学校和组织

- [ ] 可建立集团—学校—校区层级；
- [ ] 可建立学校内部部门树；
- [ ] 部门移动和停用不破坏历史任职；
- [ ] 学校支持中文名、英文名、别名、域名和课程体系；
- [ ] 学校列表支持城市、类型、课程、负责人、最近联系和关键角色缺失筛选。

## 3. 人员和任职

- [ ] 同一Person可同时是学校联系人和家长；
- [ ] 一人可有多个联系方式，每类最多一个有效主联系方式；
- [ ] 联系人转校或晋升不复制Person；
- [ ] 当前、未来和历史任职可并存；
- [ ] 任职记录包含标准职位、原始Title、部门、决策角色、影响力、关系强度和来源。

## 4. 学生和家庭

- [ ] 一个家庭可包含多个学生；
- [ ] 一个学生可有多个家长或监护人；
- [ ] 主要联系人、法定监护人、付款人和决策人可以不同；
- [ ] 学生姓名与联系方式来自统一Person；
- [ ] 学生支持A-Level、IB、AP及可配置课程；
- [ ] 学籍、课程、年级、成绩和升学目标保留历史；
- [ ] 家长沟通特征包含来源、日期和置信度。

## 5. 销售

- [ ] 学校和家庭各有独立默认Pipeline；
- [ ] Lead至少关联一个业务主体；
- [ ] Lead转化事务不会产生半成品；
- [ ] Opportunity至少关联学校、家庭或学生之一；
- [ ] Stage变化保留完整历史；
- [ ] 金额、概率、产品、决策人和下一步按阶段规则校验；
- [ ] Kanban拖拽失败时UI恢复原状态。

## 6. 活动和任务

- [ ] 同一Activity可出现在学校、联系人、家庭、学生和Opportunity时间线；
- [ ] 有效外部沟通必须创建下一步，或说明无需下一步；
- [ ] Task支持负责人、协作者、优先级、截止时间、提醒和完成结果；
- [ ] 逾期任务可筛选并通知；
- [ ] 自动任务保存来源和Workflow ID。

## 7. 导入和去重

- [ ] 支持CSV和XLSX；
- [ ] 支持字段映射、转换、验证和预览；
- [ ] 重复候选显示命中规则，不因同名直接合并；
- [ ] 每行保存CREATE、UPDATE、MERGE、SKIP或ERROR结果；
- [ ] 错误行不阻断其他有效行；
- [ ] 可生成逐行错误报告；
- [ ] 导入批次可回滚，存在后续人工修改时进入冲突处理；
- [ ] 10000行测试文件可稳定完成。

## 8. 自动升年级

- [ ] 学年日历和年级映射可配置；
- [ ] 系统先生成待升级事件，而非直接覆盖；
- [ ] 留级、跳级、转学、休学、Gap Year、提前或延迟毕业可处理；
- [ ] 同一学生同一学年重复执行保持幂等；
- [ ] 执行保存Before/After快照；
- [ ] 执行失败可重试；
- [ ] 回滚不覆盖升级后合法人工修改；
- [ ] 毕业学生进入Graduated或Alumni状态。

## 9. 报告

- [ ] 学校、学生、家庭和销售报告可生成；
- [ ] 报告数字来自数据库查询；
- [ ] AI只生成叙述草稿；
- [ ] PDF和XLSX文件使用私有存储；
- [ ] 下载链接过期；
- [ ] 下载和敏感字段导出有审计；
- [ ] 报告保存筛选和指标快照。

## 10. AI

- [ ] Provider、模型和Prompt版本可配置；
- [ ] 每次运行记录用途、模型、Token、成本、延迟和状态；
- [ ] 提取结果显示证据与置信度；
- [ ] 高风险字段必须人工确认；
- [ ] AI失败不阻断CRM核心功能；
- [ ] 自然语言查询转成可见、可编辑的筛选条件，不执行模型生成SQL；
- [ ] 用户接受、编辑或拒绝AI建议均可审计。

## 11. 非功能

- [ ] 常用列表和详情P95达到PRD目标；
- [ ] 10万Person和20万Activity测试规模下可用；
- [ ] 数据库迁移可在空库完整执行；
- [ ] 完成备份恢复演练；
- [ ] 文件和数据库恢复后关联一致；
- [ ] 无高危越权、公开文件、明文Secret或敏感日志；
- [ ] Staging和Production隔离；
- [ ] 管理员手册、导入模板和数据字典完成。
