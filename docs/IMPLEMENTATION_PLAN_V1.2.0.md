## 外部依赖说明

以下事项可以完成代码、配置验证和降级 UI，但真实生产连通需要环境所有者提供凭据：

- 邮件、日历、通知提供商。
- Turnstile 站点密钥与密钥。
- 可选 AI 提供商和显式数据处理授权。
- 生产域名与 DNS。

缺少外部凭据时，相关功能必须显示“待配置”且 readiness 能区分“必需阻断”和“可选功能未启用”。
